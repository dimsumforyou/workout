Sync logs through your own Google Sheet so phone and laptop share one history. Not connected yet? One-time setup (~3 min), no coding required:
  1. Create a new Google Sheet, then open Extensions → Apps Script
  2. Delete the sample code, paste in Code.gs from this repo
  3. Change the SECRET line near the top to your own random string — this is what goes in the "secret" field below
  4. Deploy → New deployment → type Web app → Execute as Me → Who has access Anyone → Deploy (approve the permission prompt)
  5. Copy the URL ending in /exec and paste it below, along with your secret → tap Connect
Made a change to the script later? You must redeploy for it to take effect: Deploy → Manage deployments → pencil icon → Version: New version → Deploy. Just saving the script does nothing to the live URL. Full walkthrough in the repo README.
